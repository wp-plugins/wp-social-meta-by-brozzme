Add social meta for pages in header without coding.

== Description ==
Bored about your description and images when you share your post and pages ?
Social Meta by Brozzme automaticaly add OpenGraph, Google+, Twitter metas to your header file without coding.
Meta tags in the header allow a clean share of your website pages, with good related images and description.

Options

1. Choose your social network to activate (Facebook, Google+, Twitter...)
2. Set your description, default image.
3. Automatic image array.
4. Active on index, archives pages, post, page, wooCommerce support for products pages.



== Installation ==
1. Upload "wp-social-meta-brozzme" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to Social Meta settings page.
